## How to Use:

1. **Add Your Images:**
   - Replace placeholder filenames with your actual image filenames
   - Keep images in the same folder as valentine.html
   - Format: `<img src="your-image-name.jpg" alt="Description">`

2. **Add BMW Background:**
   - Place your BMW car image file as `bmw-background.jpg`
   - Or change the filename in the CSS to match your image

3. **Customize Messages:**
   - Edit the 14-day messages array in the JavaScript section
   - Change names, dates, and personalized content as needed
   - Update birthday date (currently set to February 28)

4. **Modify Colors:**
   - Pink: `#ff4f81`, `#ff7aa2`
   - Gold: `#ffb366`
   - Change these hex codes for different color schemes
